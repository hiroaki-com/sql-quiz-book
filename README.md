# SQL Quiz Book

A collection of practical SQL quiz sets for learning through reading, thinking, and solving.

## Site

https://hiroaki-com.github.io/sql-quiz-book/

---

## Repository Structure

```plaintext
docs/
├── index.html
└── files/
    ├── sql-batch-practical-001.html
    └── ...
````

---

## Naming Convention

```plaintext
sql-{domain}-{level}-{nnn}.html
```

### Domain

* `batch`
* `api`
* `bi`

### Level

* `basic`
* `practical`

---

## Rules

* 1 HTML = 1 quiz set
* ~10 questions per file
* Numbering is independent by domain + level
* File names act as stable identifiers
